# Usage
To get started, ensure you have Docker installed on your system, then build and run the containerized services:
docker compose up --build

# Technologies:

Core Language: Python 3.14

API Framework: FastAPI (with Uvicorn)

Experiment Tracking & Registry: MLflow

Containerization: Docker & Docker Compose

# AI Disclouser
In the development of this project, generative AI models (specifically Gemini Pro and GPT-OSS 20b) were utilized as pair-programming and debugging assistants. Their primary applications included:

Generating standard boilerplate code and structural layouts.

Troubleshooting and debugging complex logic or configuration issues.

All AI-assisted code was rigorously reviewed, manually tested, and fully comprehended by the author to ensure it meets the highest quality standards and aligns perfectly with the project's parameters.
